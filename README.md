# Web Search, Extraction, and Crawling using Tavily API

This repository is a **learning project** that demonstrates how to use the **Tavily API** in Python to enable AI applications or agents to access **real-time web data**.

The project focuses on three core capabilities:
- Web search with semantic understanding
- Content extraction from live web pages
- Crawling and mapping websites for comprehensive information

These capabilities are essential when building AI agents that require **up-to-date, accurate, and context-aware information** from the internet.

2. **Content Extraction**
   - Extract clean and usable content from URLs
   - Retrieve raw page content for deeper analysis

3. **Web Crawling**
   - Crawl entire websites starting from a root URL
   - Collect linked pages and their content

4. **Website Mapping**
   - Map a website structure
   - Use natural language instructions to guide what information should be extracted
